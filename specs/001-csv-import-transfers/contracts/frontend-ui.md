# contracts/frontend-ui.md

## UI Contracts

### Sidebar
- Collapsible left sidebar with menu items
- Bottom sign-in/sign-out icon

### Topbar
- Back button, title, description, custom action buttons (e.g., import CSV)

### Transfer Page
- Paginated, sortable, filterable list of transfers
- Action buttons for manual labeling

### Bank Account Page
- Paginated, sortable, filterable list
- Action buttons for linking labels/regexes

### Label Page
- Set max values/percentages
- Visualize remaining amount for period

### Statistics Page
- Select date range and labels
- Visualize in circle diagrams and bar charts

### CSV Upload
- Progress bar, error messages, success feedback

### Authentication
- OAuth2 sign-in/sign-out

### Debugging
- Show import steps and errors clearly
